# Pinecone Web Search

This project is designed to efficiently store, manage, and search vector data using Pinecone.

## Features
Stores and retrieves vectors efficiently  
Supports semantic search  
Uses Pinecone's indexing for fast lookups  

## Installation
Follow these steps to set up the project locally:

1. Clone the repository:
   ```bash
    git clone https://github.com/NamitaChougule17/Pinecone-Web-Search

2. Navigate to the project folder:
    cd Pinecone-Web-Search

3. Install dependencies:
    pip install -r requirements.txt
